# Password Strength Analyzer & Cracker Simulator

A powerful, dual-language security tool designed to analyze password entropy and simulate real-world brute-force attacks. This project features a custom dark-themed GUI and provides a visual, educational breakdown of how hackers crack passwords and how multi-hash generation works.

---

## 🚀 Features

* **Password Entropy Analysis:** Accurately calculates password strength based on length, character variety, and complexity.
* **Live Brute-Force Simulator:** A visual display showing how a brute-force attack cycles through combinations in real-time.
* **Multi-Hash Generation:** Instantly generates MD5, SHA-1, and SHA-256 hashes for any input string.
* **Modern GUI:** A custom, user-friendly dark-themed interface for a smooth user experience.
* **Dual-Language Core:** Leverages both Python and Java to handle computations and simulate environments efficiently.

---

## 🛠️ Built With

* **Python** - Core logic, brute-force simulation, and scripting.
* **Java** - Backend processing and structure.
* **Java Swing** - Custom dark-themed user interface.

## ⚙️ Installation

To get this project running locally, follow these steps:

### Prerequisites
Make sure you have the following installed on your system:
* Python 3.13.1 or higher
* Java Development Kit (JDK) 11 or higher

### Setup
1. **Clone the repository:**
```bash
git clone https://github.com/Aryansh1727/Password-Strength-Analyzer-Cracker-Simulator.git 
```
2. **Navigate to the project directory:**
```bash
cd Password-Strength-Analyzer-Cracker-Simulator
```
**🚦 Usage**
Depending on which version or component of the project you want to run, use the corresponding commands below in your terminal:

To run the Python Application:
```bash
python analyzer.py
```
To run the Java Application:
```bash
javac PasswordApp.java
java PasswordApp
```
